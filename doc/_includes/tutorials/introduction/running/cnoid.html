Pre-requisites

To run this interface you need to have installed {% link mc_openrtm %} on your system. Furthermore, you will need {% link Choreonoid %} and a related simulation file for your robot. An example is provided for JVRC-1.

Running the controller

Open the simulation file with Choreonoid. For example, using the JVRC example provided with {% link mc_openrtm %}:

{% highlight bash %} # Go to the directory where the simulation file has been installed $ cd /usr/local/share/hrpsys/samples/JVRC1 # Open the simulation file $ choreonoid sim_mc.cnoid {% endhighlight %}

Then click the button to start the simulation in Choreonoid. This will also start your controller. Note that you can directly start the simulation from the command line:

{% highlight bash %} $ choreonoid sim_mc.cnoid --start-simulation {% endhighlight %} {% include tutorials/introduction/running/feedback.html %}

Note: this won't enable 3D visualization beyond what is available in Choreonoid
