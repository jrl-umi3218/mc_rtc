Detailed Description

template<typename GetT, typename SetT>
struct mc_rtc::gui::XYThetaImpl< GetT, SetT >

An XYTheta element represents an oriented point in the XY plane.

Altitude can be provided optionally.

This element is editable

Template Parameters
GetTShould return a double array of size 3 (x, y, theta) or 4 (x, y, theta, z)
