Detailed Description

Configuration object that keeps track of the file it comes from

This utility saves you from keeping track of the loaded file, i.e. insteaf of

std::string path = "...";
mc_rtc::Configuration config(path);
// Work with config
config.save(path);

You can simply do:

mc_rtc::ConfigurationFile config(path);
// Work with config
config.save();
