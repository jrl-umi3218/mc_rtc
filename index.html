Key features

Simple tools for complex problems
Design a complex robotic applications on complex robots using simple graphical tools.
Extensible interface
Write new components to solve your specific problem and integrate with mc_rtc using our lean C++/Python API.
Write once, run everywhere
Run the same code in simulator(s) or on the robot. Use one of the existing interface or create your own.
Use your bot
Use your own robot in the framework, a single URDF file is enough to get you started. You can also use one of the available robots.
